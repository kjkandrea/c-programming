# 선택 제어문과 반복 제어문

* 순차적 제어 : 특별한 지정이 없는 한 위에서 아래로 수행되는 제어 구조
* 선택적 제어 : 주어진 조건에 따라 **특정 부분으로 수행을 옮기는** 분기 제어 구조
* 반복적 제어 : 특정 부분을 **일정한 횟수만큼 반복 수행하는** 반복 제어 구조
